# Library Files Required

This extension needs two JavaScript libraries in this folder:

## 1. jsPDF (for PDF generation)
Download: https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js
Save as: `jspdf.umd.min.js`

## 2. docx.js (for Word generation)
Download: https://unpkg.com/docx@8.5.0/build/index.umd.js
Save as: `docx.min.js`

## Quick Setup
Run the setup script: `./setup.sh`
Or manually download the above files into this `lib/` folder.
